## Remove exclamation marks

Write function RemoveExclamationMarks which removes all exclamation marks from a given string.
---

### Given Code


```python
def remove_exclamation_marks(s):
    pass
```

---

### Solution


```python
def remove_exclamation_marks(s):
    return s.replace("!", "")
```
