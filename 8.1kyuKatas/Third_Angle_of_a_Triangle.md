## Third Angle of a Triangle


You are given two angles (in degrees) of a triangle.

Write a function to return the 3rd.

Note: only positive integers will be tested.

---

### Given Code


```python
def other_angle(a, b):
    pass
```

---

### Solution


```python
def other_angle(a, b):
    return 180 - (a + b)
```
